### 11.2 Calibration Error 개선 방법

**현재 문제점 (90일 기준):**

| 시나리오 | 예측 평균 | 실제 발생률 | 오차 |
|----------|----------|------------|------|
| Bullish | 17.6% | 29.0% | -11.4% (과소) |
| Sideways | 61.6% | 27.7% | **+33.9%** (과대) |
| Bearish | 20.8% | 43.3% | **-22.5%** (과소) |

**개선 방안:**

1. **Platt Scaling 적용**
   ```python
   # 로지스틱 회귀로 확률 보정
   calibrated_prob = 1 / (1 + exp(-(A * raw_prob + B)))
   # A, B는 검증 데이터에서 학습
   ```

2. **Isotonic Regression**
   - 비모수적 방법으로 단조성 유지하며 확률 보정

3. **시나리오 임계값 재조정**
   ```python
   # 현재: Sideways = abs(return) < 5%
   # 개선: 변동성 기반 동적 임계값
   sideways_threshold = historical_volatility * 0.5
   ```

4. **시장 상태별 분리 보정**
   - 시장 상태(19개)별로 별도 calibration 계수 적용
